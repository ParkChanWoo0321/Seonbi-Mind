# 선비마인드

## 📌 서비스 소개

**선비마인드**는 고전 문장, 사자성어, 고사성어, 전통적인 우리말 표현을 쉽고 재미있게 학습할 수 있도록 만든 **게임형 고전 언어 학습 서비스**입니다.

디지털 환경의 확산으로 짧고 자극적인 콘텐츠 소비가 증가하면서 줄임말과 신조어 중심의 언어 사용이 일상화되고 있습니다. 이러한 변화 속에서 고전 문장과 전통 언어 표현은 점차 일상에서 접하기 어려워지고 있으며, 자연스럽게 학습할 수 있는 기회도 줄어들고 있습니다.

선비마인드는 이러한 문제를 해결하기 위해 고전 언어 표현을 현대적으로 재해석하고, 퀴즈와 AI 피드백을 결합하여 누구나 부담 없이 학습할 수 있는 환경을 제공합니다.

---

## 🎯 기획 배경

현재 영어 학습 서비스, 단어 암기 앱, 게임형 언어 학습 플랫폼은 다양하게 존재합니다. 그러나 우리 고유의 고전 언어 표현을 쉽고 재미있게 학습할 수 있는 디지털 서비스는 상대적으로 부족합니다.

기존의 고전 언어 학습은 일부 학습자에게만 제한적으로 소비되는 경우가 많고, 일반 사용자가 일상 속에서 활용하기에는 접근성이 낮습니다. 장기적으로는 이러한 언어 자산이 점차 활용되지 못하고 사라질 가능성도 있습니다.

따라서 선비마인드는 고전 우리말을 현대적인 학습 콘텐츠로 재구성하여, 사용자가 짧은 시간 안에 반복적으로 학습하고 자연스럽게 언어 표현력을 높일 수 있도록 설계되었습니다.

---

## 👥 주요 사용자

### 사회 초년생 및 직장인

보고서 작성, 발표, 커뮤니케이션 과정에서 보다 정확하고 품격 있는 표현을 필요로 하는 사용자입니다. 실무에서 활용 가능한 어휘력과 표현력을 높이는 데 도움을 줄 수 있습니다.

### 자기계발에 관심 있는 사용자

짧은 시간 안에 교양, 어휘력, 문해력, 표현력을 함께 향상시키고자 하는 사용자입니다. 반복 학습 기반의 콘텐츠 소비 성향과 높은 적합성을 가집니다.

### 고전 및 전통 언어에 관심 있는 사용자

고사성어, 사자성어, 고전 문장 등을 깊이 있게 학습하고자 하는 사용자입니다. 단순 암기를 넘어 표현의 의미와 활용 맥락을 이해할 수 있습니다.

### 중장년층 사용자

옛 우리말과 전통 표현에 관심이 있는 사용자에게도 의미 있는 학습 경험을 제공합니다. 익숙한 언어 자산을 디지털 환경에서 다시 접할 수 있도록 돕습니다.

---

## 🧩 주요 기능

## 1. 단계형 퀴즈 학습

사용자는 단계별 학습 코스를 선택하여 총 4문제로 구성된 퀴즈를 진행합니다.

각 문제는 고전 문장, 사자성어, 고사성어, 전통 표현을 기반으로 출제되며, 짧은 시간 안에 학습할 수 있도록 구성됩니다.

이를 통해 사용자는 부담 없이 반복 학습을 수행하고, 자연스럽게 고전 표현을 익힐 수 있습니다.

---

## 2. AI 훈장님 피드백 시스템

선비마인드는 서당의 훈장님 캐릭터를 활용한 AI 피드백 시스템을 제공합니다.

사용자가 답변을 제출하면 정답 또는 오답 여부를 판단하고, AI 훈장님이 해당 답변에 대한 해설을 제공합니다.

단순히 정답만 알려주는 것이 아니라, 왜 맞았는지 또는 왜 틀렸는지를 설명하여 사용자가 학습 내용을 즉시 이해할 수 있도록 합니다.

---

## 3. 고전 표현 해설 기능

문제 풀이 후 해당 표현의 의미와 활용 맥락을 현대 언어로 설명합니다.

이를 통해 사용자는 고전 표현을 단순히 암기하는 것이 아니라, 실제 의미와 사용 상황을 함께 이해할 수 있습니다.

---

## 4. 반복 학습 구조

짧은 문제 구성과 단계별 진행 방식을 통해 사용자가 부담 없이 반복 학습할 수 있도록 설계하였습니다.

반복적인 문제 풀이를 통해 어휘와 표현이 자연스럽게 축적되며, 학습 부담을 줄이고 꾸준한 학습 습관 형성을 유도합니다.

---

## 5. 레벨 기반 성장 시스템

사용자의 지속적인 학습 참여를 유도하기 위해 게임형 레벨 성장 구조인 **문패 시스템**을 도입하였습니다.

퀴즈를 수행하면 경험치가 축적되고, 일정 기준을 달성하면 단계별로 승급할 수 있습니다.

| 레벨 | 명칭 |
|---|---|
| 1 | 도사 |
| 2 | 정랑 |
| 3 | 첨정 |
| 4 | 사인 |
| 5 | 집의 |
| 6 | 참의 |
| 7 | 동지사 |
| 8 | 지사 |
| 9 | 판사 |
| 10 | 영의정 |

최종 단계인 **영의정**에 도달하면 프로필 이미지 변경과 같은 보상 요소를 제공하여 사용자의 성취감을 높이고 지속적인 서비스 이용을 유도합니다.
